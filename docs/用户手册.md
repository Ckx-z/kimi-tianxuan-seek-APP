# COF 成膜推荐系统 · 用户手册

> 面向实验人员的操作手册。不需要任何编程知识，按页面照着点即可。
> 安装包安装后，从开始菜单或桌面快捷方式启动「COF成膜推荐系统」。首次启动约 5–8 秒属正常。

---

## 一、查询打分（首页）

预测一对"醛单体 + 胺单体"组合的成膜概率。

1. 在「醛单体」「胺单体」输入框中任选一种方式输入：
   - **SMILES**：直接粘贴，例如 `O=CC1=CC=CC=C1C=O`
   - **CAS 号**：直接输入，系统自动换算结构
   - **内置单体库**：点下拉列表按名称/类别挑选
2. 点击「打分」，稍等 1–2 秒，结果区显示：
   - **成膜概率 / 分数**：0–1 之间，越高越可能成膜
   - **不确定度**：分数后面的 ± 范围，范围大说明模型把握小
   - **OOD 标记**：
     - 无标记 = 在训练分布内，分数可信
     - ⚠️ = 外推警告（单体超出训练分布），分数仅供参考
     - ⛔ = 非标准官能团，不出分（不要强行解读）
   - **打分理由**：中文解释哪些官能团/特征推高或拉低了分数
   - **化学结构图**：醛、胺及缩合产物骨架的 2D 结构
   - **相似成膜案例**：历史上最接近的真实成膜组合
3. 看到满意的组合，点「⭐ 收藏」加入收藏夹。

> GNN 分量显示"不可用"不是故障，见 FAQ。

## 二、批量排序

一次比较多组单体组合。

1. 进入「批量排序」页，按模板逐行粘贴多组（醛, 胺）SMILES 或 CAS，每行一组。
2. 点击「批量打分」，结果按成膜分从高到低排列，含 OOD 标记与不确定度。
3. 点「导出 CSV」保存结果表，可用 Excel 打开做进一步筛选。
4. 排序表中的组合可直接勾选加入收藏夹。

**建议用法**：先用批量排序筛出前 5–10 组，再逐组到首页看打分理由与相似案例。

## 三、实验记录

把真实实验结果回填，形成"预测 vs 实际"对照，也是方案迭代的依据。

1. 进入「实验记录」页，点「新建记录」。
2. 填写：
   - 关联的预测组合（可从收藏夹/历史预测中选择）
   - **实验条件**：溶剂、温度、时间、催化剂等
   - **实际结果**：成膜 / 部分成膜 / 不成膜 + 备注（现象描述越具体越好）
3. 保存后，记录出现在时间线中，可与当时的预测分数对照。
4. 记录支持**删除**和**放大查看**详情；可生成 Word 实验报告（内嵌结构图）。

> 认真填写失败记录同样重要——GraphRAG 迭代引擎会利用成功与失败案例给出改进建议。

## 四、收藏夹

- 收藏的组合自动**匹配文献**（显示真实论文标题），方便查原始出处。
- 可随时取消收藏；收藏内容只存在你自己的电脑上。

## 五、方案迭代（GraphRAG）

用自然语言和系统讨论"下一步怎么做"。

1. 进入「方案迭代」页，像聊天一样提问，例如：
   - "这个组合成膜失败了，可能是什么原因？"
   - "醛单体带 F 取代的组合，文献里常用的成膜条件是什么？"
   - "帮我把上一组实验的条件往高温方向调整，给个具体方案。"
2. 系统先在知识图谱中检索相关文献与实验记录，再给出建议（附取证依据）。
3. 觉得建议靠谱，点「采纳」——自动生成一张**带编号的方案卡**，可导出、可存档，下次实验照卡执行。
4. 方案卡和迭代历史只存在本机。

## 六、设置页：配置 LLM

LLM 用于单体性质卡、实验方案卡与方案迭代问答。**不配置不影响打分、排序、收藏、记录等核心功能。**

1. 打开「设置」页，填写三项：
   - `base_url`：API 端点，如 `https://api.longcat.chat/openai/v1`
   - `api_key`：你的密钥
   - `model`：模型名，如 `LongCat-2.0`
2. 支持任何 OpenAI 兼容接口（MiniMax / LongCat / OpenAI / 本地 vLLM）。向课题组管理员或负责人索取可用账号。
3. 保存后写入本机 `%APPDATA%\COF-Film-Recommend\config\llm_settings.local.json`（仅本机，不随安装包分发）。
   也可手动编辑该文件，或用环境变量 `COF_LLM_BASE_URL / COF_LLM_API_KEY / COF_LLM_MODEL`。

## 常见问题 FAQ

**Q1：双击后打不开 / 一直转圈？**
- 首次启动需 5–8 秒加载模型，请稍等；超过 30 秒仍无反应再继续排查。
- 检查杀软是否拦截：把安装目录（含 `_internal` 子目录）加入白名单后重试。
- 查看 Windows 任务管理器中是否已有残留进程，结束后再启动。
- 8000 端口被占用**不影响**——应用会自动选择其他端口。

**Q2：杀毒软件报毒或首启很慢？**
安装包内 `_internal` 目录含大量 Python 运行文件，杀软扫描会明显拖慢首启，甚至误报。将安装目录整体加入杀软白名单即可。安装包为课题组内部构建，可放心添加。

**Q3：界面提示"GNN 不可用"，是坏了吗？**
不是。GNN 是可选增强模型，需要额外的 GPU/PyTorch 环境；没有该环境时系统自动降级为仅 tree 模型打分。**tree 模型是主力模型，分数完全可用**，只是少了 GNN 分量对照。

**Q4：我的数据存在哪里？会丢吗？**
全部在本机 `%APPDATA%\COF-Film-Recommend\`（收藏、实验记录、方案卡、LLM 配置）。卸载软件时可选择保留；重装或换电脑时拷贝该目录即可迁移。

**Q5：打分显示 ⚠️ 或 ⛔ 是什么意思？**
- ⚠️ 外推警告：单体超出训练分布，分数仅供参考，建议结合相似案例与文献判断。
- ⛔ 非标准官能团：该单体不属于系统支持的醛/胺体系，不出分。

**Q6：SMILES 输进去报错？**
检查是否复制完整、无多余空格/换行；或改用 CAS 号 / 内置单体库输入。

**Q7：LLM 相关按钮点了没反应？**
先到「设置」页确认 LLM 三项已正确填写且密钥有效；网络需能访问对应 API 端点。
