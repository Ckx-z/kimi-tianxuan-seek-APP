## 一、什么是 COF（共价有机框架）

**共价有机框架（Covalent Organic Frameworks, COFs）** 是一类由轻元素（C、H、O、N、B 等）通过强共价键连接而成的多孔晶体聚合物。

核心特点：
- **有序多孔结构**：像分子筛一样有规整的孔道
- **全有机**：不含金属（区别于 MOF）
- **可设计性**：通过选择单体（building blocks）定制结构和功能
- **应用**：气体分离、催化、传感、储能、光电等

## 二、COF 的成膜问题

COF 有两种主要产物形态：

| 形态 | 特点 | 应用 |
|---|---|---|
| **粉末（powder）** | 不连续的固体颗粒 | 催化、吸附、传感 |
| **薄膜（film/membrane）** | 连续致密的膜层 | 分离膜、器件 |

**成膜比成粉难得多**——这正是本项目的核心难题。能成膜的 COF 才能做分离膜等高价值应用，相关研究更容易发表。

## 三、COF 成膜的化学原理（亚胺键路线）

本项目聚焦最常见的 **Schiff base（席夫碱）反应**路线：

```
醛（aldehyde, -CHO） + 胺（amine, -NH₂） → 亚胺键（imine, -CH=N-） + H₂O
```

典型单体组合：
- **醛单体**：如 Tp（1,3,5-三甲酰基间苯三酚）、TFB（1,3,5-三(4-甲酰基苯基)苯）
- **胺单体**：如 Pa（对苯二胺）、BD（联苯胺）、TAPB（1,3,5-三氨基苯）

### 拓扑规则（为什么有的组合能成膜）

单体的**官能团数量和位置**决定能否形成二维网络：
- **C2 对称单体**（2 个反应位点，对位）：如对苯二胺 → 线性链
- **C3 对称单体**（3 个反应位点，120° 分布）：如 Tp、TFB → 网状结构
- **拓扑匹配**：C2+C2（链）、C3+C2（网）、C3+C3（网）等组合才可能成膜
- **官能团位置**：必须是对位（para）或合适的间位（meta），邻位（ortho）通常不行

### 影响成膜的关键因素

1. **单体结构**：芳香性、平面性、对称性、官能团匹配
2. **反应条件**（本项目重点）：
   - **界面类型**：液-液界面法最常见（水/有机溶剂两相）
   - **溶剂**：水/二氯甲烷、水/乙酸乙酯等
   - **催化剂**：乙酸（最常见）、醋酸水溶液
   - **温度**：室温到 120°C
   - **浓度**、**时间**
3. **氟取代**（旧项目重点关注）：F、CF3 取代基影响成膜（旧项目发现 F-醛预测偏低）

## 四、为什么用机器学习

COF 单体空间巨大（数百种醛 × 数百种胺 = 数万种组合），穷举实验不可行。ML 能：
- 从文献积累的实验数据中学习"什么组合+什么条件能成膜"
- 对未试过的组合打分排序，指导实验师优先试哪些
- 给出可解释的理由（SHAP：哪些因素促进/阻碍成膜）

## 五、核心挑战（本项目要解决的）

1. **发表偏差**：能成膜的才发表 → 文献里几乎全是正样本 → 负样本严重缺失且不可信
2. **条件敏感**：同一组合不同条件可能成膜也可能成粉 → 不能脱离条件谈结构
3. **样本量小**：去重后独立 COF 组合可能只有几百个 → 深度学习易过拟合
4. **标签噪声**：LLM 从文献提取的标签可能有误
