**数据层和诊断能力是强项，瓶颈在特征利用和评估方法，不在模型架构。**

## 做得好的

1. **数据工程是全项目最亮的环节**：1719篇PDF→6201条训练数据，v5假负样本修复让PR-AUC +81%
2. **问题诊断能力远超平均水平**：2026-05-21系统性缺陷复盘（8大类）非常深刻
3. **化学领域知识编码很用心**：34维规则向量、26维linker描述符、4种负样本策略
4. **工程化程度高**：15-fold CV、MC Dropout、频率降权、4步筛选管线
5. **实验记录做得好**：jiyi/目录的阶段记录和版本演进

## 不够好的

1. **GNN是过度选择**：XGBoost基线0.654时描述符未归一化，RF失败被误归因为"不适合"
2. **反应条件从未使用**：v5_train_stage1.csv已有6个条件字段但GNN完全忽略
3. **筛选和训练问题混淆**："泛化差"主要在筛选阶段（笛卡尔积外推），非训练CV
4. **增广策略方向有误**：200条增广在6000样本上改变不了分布

## 被低估的资产

1. **v5数据集本身**：6201条+四分类+34维规则+954篇YAML，最好的COF成膜数据集之一
2. **GNN 0.784是强基线**：新模型需明确超越
3. **知识图谱**：809单体+954文献+1882边，可用于相似案例推荐
4. **化学直觉驱动改进**：每次提升本质是人的判断纠正模型，SHAP可验证"模型学到了什么"

---

## 新项目的核心价值

不是"换模型"，而是：
1. **用条件特征释放已有数据的潜力**（最大增量）
2. **用留一单体法验证真正的泛化**（旧项目未做）
3. **用SHAP让模型学到化学规律**（替代人工驱动）
4. **用推荐指标（NDCG/Hit@K）匹配实验需求**（替代二分类PR-AUC）
