## 傍晚追加：GitHub Release 发布 + README 快速上手指引

### Release 凭证问题定性

- GitHub 官方 MCP 本身**不具备** Release/资产上传工具（非用户授权范围问题），重连插件无用
- 用户提供了 fine-grained PAT（Contents RW，仅限本仓库）→ 走 REST API 完成全部操作

### 网络坑与解法（重要环境知识）

- **Watt Toolkit（进程名 Steam++/Steam++.Accelerator）把 GitHub 域名写入 hosts → 127.0.0.1**，本地 443 反代加速
- curl 默认被 127.0.0.1 拦下（证书问题瞬断 000）；`--resolve` 绕过则只有 6KB/s（直连 GitHub 被限速）
- **正解：curl 加 `-k` 走 Watt 加速通道** → 上传 2.6MB/s（357MB ≈ 142s）
- SSH push 间歇被 Watt 重置（Connection reset port 443）→ **备用通道：`GIT_SSL_NO_VERIFY=1 git push https://x-access-token:PAT@github.com/...`**，一次成功
