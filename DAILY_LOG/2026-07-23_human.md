## 阻塞点

- **GitHub Release 发布**：你仓库的 git 走 SSH 密钥、GitHub 插件授权不含"创建 Release"权限，我这边传不上去。三个解法任选：① 给我一个 PAT（GitHub → Settings → Developer settings → Tokens，勾 repo 权限）；② 在 Kimi 里重连 GitHub 插件并授权更大范围；③ 你在仓库网页点 Releases → Draft a new release，选 v0.1.0 标签把 zip 拖进去（标签我已经推好了）。
- **Mac 版本**：目前只有 Windows 版。Mac 需要一台 Mac 电脑重新打包（或后续用 GitHub Actions 云端自动打，技术上可行，需要时我来做）。
