# 失败判据手册 (Class A-G)

本手册定义 7 档失败分类的统一判据、可能根因、应对建议。  
每条实验反馈必须落到一档，根因 Type 作为辅助标签（不强制）。

---

## 总体分类逻辑

```
A (无产物) → B (未充分反应) → C (无定形) → D (弱结晶) → E (膜质量差) → F (膜质量中) → G (膜质量高)
   递进关系: 越往后越接近成功。Class 越低越需要从根因入手；越高越需要精修条件。
```

---

## Class A — 无产物

**判据**（48-72h 反应后）  
- 反应液为**清液 / 微浊液 / 凝胶 / 油状**，无可见固体  
- 玻璃壁无膜产出  
- 瓶底无沉淀

**可能根因**（Type 单体 / 条件 / 操作 三选一或并列）  
1. Type 单体 — 单体本身在该条件下无法反应（活性不匹配、电子效应不兼容）
2. Type 条件 — 温度未达 / 催化剂浓度不足 / 反应时间不够
3. Type 操作 — 加料顺序错（苯胺未先于二胺接触醛基）

**应对建议**  
- 先做 PXRD 确认是否有微量产物（可能<检测限）  
- 检查加料顺序日志，对照"通用步骤"  
- 若加料无误：升温 +10°C / 增加乙酸浓度 / 延长至 96h 重试

---

## Class B — 未充分反应

**判据**  
- 反应液含**大量未溶单体**（肉眼可见固体原料残留）  
- 上层清液 + 底部沉淀分层明显  
- 玻璃壁无膜

**可能根因**  
1. Type 单体 — 单体溶解度过低（特别见于大骨架、高 LogP 单体如 H4、A7）
2. Type 条件 — 溶剂选择不当 / 温度未达单体溶解所需 / 配比超出溶解窗口

**应对建议**  
- 查 `实验/方案/实验方案_含氟COF薄膜合成_v3.9_20260626.docx` §2.5 单体溶解度表  
- 切换到该单体溶解度+的溶剂体系  
- 必要时预溶（60-80°C 超声 30 min）  
- 若 H 系列：BTF 比例需提高

---

## Class C — 无定形产物

**判据**  
- 形成**固体粉末 / 凝胶 / 沉淀**  
- PXRD 无峰或馒头峰（典型 2θ 范围 10-30° 宽包）  
- 玻璃壁可能有少量附着但非膜

**可能根因**  
1. Type 单体 — 单体反应活性过高（CF3/F 邻位、位阻小）→ 不可逆聚合
2. Type 条件 — 苯胺调制量不足 / 催化剂过浓 → 动力学过快

**应对建议**  
- 增加苯胺量 1-2 eq（封端醛基增强可逆性）  
- 降低乙酸浓度 50%（如 6.0M → 3.0M）  
- 温度降至 60-80°C 减缓动力学  
- 考虑分次加催化剂（0.1 mL + 0.1 mL 间隔 1h）

---

## Class D — 弱结晶产物

**判据**  
- 粉末形成，PXRD 有**弱衍射峰**（fwhm 大、强度低、峰位不锐利）  
- 未形成连续膜  
- 可能伴随多相混合

**可能根因**  
1. Type 条件 — 结晶时间不足 / 温度未达最佳结晶窗口
2. Type 单体 — 单体对称性差 / 杂质干扰

**应对建议**  
- 延长反应时间至 96-120h  
- 考虑两段控温（72h 60°C 慢反应 + 48h 80°C 加速结晶）  
- 配合 BET 数据判断比表面积是否达标

---

## Class E — 膜形成但质量差

**判据**（这是 ABCDEF 实验 A/B 的典型情况）  
- 玻璃壁**有膜形成但不连续 / 粗糙 / 易碎 / 多层 / 漂浮**  
- PXRD 可能显示一定结晶性但与目标不符  
- 膜厚度不均匀

**可能根因**  
1. Type 条件 — 苯胺调制量与单体不匹配 / 温度过高 / 时间不够
2. Type 单体 — 骨架扭转（如 F/CF3 邻位联苯）→ 平面性差

**应对建议**  
- 延长反应时间至 72-96h  
- 苯胺维持 3 eq 起步，按需±1 eq  
- 玻璃管酸处理活化（参考 `实验方案_壁面预涂覆策略` v1.0 §4.2.1）  
- 考虑壁面预涂覆（高级策略）

---

## Class F — 膜质量中

**判据**  
- **连续膜**已形成（覆盖面积 ≥50% 玻璃壁反应区）  
- PXRD 有峰但 fwhm > 0.20° 或强度一般  
- FTIR 确认亚胺键 C=N ~1631 cm⁻¹ 形成  
- SEM 显示连续但晶面取向不完美

**可能根因**  
- 接近最优，需精修

**应对建议**  
- 微调温度 ±5°C  
- 微调苯胺 ±0.5 eq  
- 延长 24h 看 fwhm 是否收敛  
- 可考虑做 BET 测比表面积

---

## Class G — 膜质量高

**判据**  
- 连续膜覆盖面积 ≥80%  
- PXRD fwhm ≤ 0.16°（参考 AMCOF-1）  
- FTIR 亚胺键完全形成，残余醛基 <5%  
- SEM 显示晶面取向良好、气相/玻璃侧差异正常  
- BET 比表面积达标（如 >800 m²/g）

**可能根因**  
- 成功，无须调整

**应对建议**  
- 重现性验证（≥3 次重复）  
- 记录完整条件进 `feedback_db.csv` 作为后续新方案的相似案例  
- 可作为新方案的阳性对照

---

## 根因 Type 辅助标签（非强制）

| Type | 含义 | 处置 |
|------|------|------|
| **单体** | 单体本身在该条件下无法成膜 | 反馈进 tianxuan seek 黑名单（可选） |
| **条件** | 单体对但条件不匹配 | 反馈进条件库，迭代下一轮方案 |
| **操作** | 加料/转移/温度控制失误 | 改进 SOP，不进模型训练 |

---

## PXRD 必填规则

- Class A / B — 必空（无固体可测）
- Class C / D / E / F / G — **强烈建议附 PXRD 文件路径**（在 `feedback_db.csv` 的 `PXRD文件` 列）
