# 失败应对 Playbook

本文件按"预测失败模式 → 调整方案"组织每个进行中实验的应对策略。
配合 `failure_criteria.md` 使用 — 那是 Class A-G 通用判据，这里是具体实验的处方。

---

## 全局观察 (影响所有进行中实验的 3 个共同问题)

### 问题 1：**膜反复溶解-再形成**（自反应法通用）
- **观察**：A1, A2, A5, D3 在 7/6 重启加热时都观察到膜溶解、缩小、再形成
- **共同根因猜测**：
  1. 温度波动冲刷已形成膜
  2. 溶剂回流溶解
  3. 苯胺/乙酸二次加入时局域浓度过高破坏成核位点
- **统一应对**：
  - **不再中途重启** — 一次性加热 120°C × 72h 不停
  - **降低苯胺/酸冲击** — 加醛前先降温到 80°C 或室温，再分 3-4 次滴加苯胺+酸
  - **降低反应温度** — 尝试 80-100°C 长时 (96-120h) 代替 120°C 短时

### 问题 2：**粉红色油状物**（TFPT 节点通用）
- **观察**：D3, D7, D9 (含 TFPT) 都观察到液面有**粉红色油状物**
- **共同根因猜测**：
  - TFPT 与胺反应过快形成**低聚物/寡聚物**（不是 COF 骨架）
  - 油状物是这些低聚物聚集析出
  - 在膜形成过程中起干扰作用（占用了胺基反应物）
- **统一应对**：
  - **降低催化剂浓度** — 12M/6M 乙酸降到 3.0M 或更低
  - **增加苯胺量** — 提到 4-5 eq 提供更多可逆缓冲
  - **降温** — 80°C 反应速率更可控
  - **先混合再加热** — 避免局域高浓度

### 问题 3：**加料顺序对动力学的影响**（跨实验对照）
- **3 种加料顺序正在同时验证**：
  - 标准序 (D7): 先醛+苯胺后胺+酸
  - 自反应法 (A1, A2, A5, D3): 先节点 12h 后加第二种单体
  - 单次加料 (A8, D9): 先醛+苯胺→立即胺→最后酸
- **建议**：等这批结果出来后做 **3-way 对照**总结（不是当下立即行动）

## A1 (TAPT + A6, 0.819, 自反应法)

### 当前已知状态
- 12h 壁上形成**连续光滑膜**（最优）
- 反复溶解-再形成（7/6-7/8）
- 7/8 加醛后 52-72h **管壁连续但粗糙**
- **液面有粉红色油状物** + 底部固体沉淀

### 预测失败模式与应对

#### 失败 α：**72h 时膜不连续**（最差情况）
- **判据**：72h 管壁仅有零星固体，无连续膜区
- **根因**：TAPT 自反应 12h 后形成的初始膜在 7/6 重启时已部分溶解；加醛后无法重新良好结晶
- **应对迭代**：
  ```
  方案 A1-v2 改进:
  - 一次性加热 120°C × 72h 不停 (不重启，不冷却)
  - TAPT 自反应缩短到 6h (而不是 12h)
  - 加醛时管温先降到 80°C，避免局域冲击
  - 苯胺量提高: 8.2 μL → 13.7 μL (按 v3.9 G3-S2 当量)
  - 延长总反应时间: 72h → 96h
  ```

#### 失败 β：**72h 时膜连续但粗糙**
- **判据**：管壁有连续膜但表面颗粒感强，不适于分离/传感应用
- **根因**：苯胺调制不足或催化剂浓度过高 → 结晶过快
- **应对迭代**：
  ```
  方案 A1-v2:
  - 苯胺提到 13.7 μL (1:1 醛当量)
  - 乙酸降到 3.0 M (从 6.0 M)
  - 保持 120°C × 72h
  ```

#### 失败 γ：**粉红色油状物干扰成膜**
- **根因**：可能是 TFPT 三嗪核体系特有（A1 用 TAPT 也出现，可能是酰肼类或 CF3 取代副反应）
- **应对**：
  ```
  - 加醛后用分液漏斗吸出粉红色油状物
  - 或降低整体温度到 80°C
  ```

## A2 (TAPB + A6, 0.819, 自反应法 + TAPB 对照)

### 当前已知状态
- 12h 壁上形成**不连续、非常粗糙类膜**（与 A1 同样条件但 TAPB 替代）
- 加醛后 52-72h **仍不连续、粗糙深褐色固体**

### 预测失败模式
- **大概率失败**（基于 A1 的 12h 形成光滑膜对比，A2 同条件仅不连续）
- 失败 Class 预期：**D** （弱结晶产物）或 **E** （膜不连续粗糙）

#### 失败 α：**膜完全不连续**
- **应对**：
  ```
  - 直接放弃 TAPB，迁移到 TAPT 体系（v3.9 已切换）
  - 后续设计不再含 TAPB 实验组
  ```

#### 失败 β：**形成弱结晶粉末**
- **应对**：
  ```
  - 同 A1 应对迭代 (按苯胺/乙酸/温度调整)
  - 但优先考虑改中心核到 TAPT
  ```

---

## A5 (TAPT + A6, 0.819, 自反应法 + 氯仿变种)

### 当前已知状态
- 12h 壁上**连续但粗糙颗粒堆积**
- 52h 后 **不连续粗糙褐色固体**
- 72h 后**逐渐变连续、光滑**（最有希望）

### 与 A1 唯一差别
- A1 加 A6 时用 甲苯 0.12 mL + 氯仿 0.88 mL
- A5 加 A6 时用 甲苯 0.12 mL + **氯仿 1.08 mL**（多 0.20 mL）

### 预测失败模式
#### 失败 α：**氯仿比例放大未改善** (72h 仍粗糙)
- **应对**：
  ```
  - 退回 A1 原始 0.88 mL 氯仿
  - 不再氯仿变种
  ```

#### 成功 β：**72h 渐变连续光滑** (观测到)
- **动作**：72h 取样做 XRD、SEM、FTIR 表征，看是否达到 Class F/G
- **应对**：若 Class F/G，转入重现性验证

---

## A8 (TAPT + A6, 0.819, 新加料顺序)

### 当前已知状态
- **方法**：先 A6 + 苯胺 → 立即加 TAPT → 最后加酸
- 28h: 液面上方**少量固体堆积形成的黄色薄膜，比较连续、粗糙**
- 46h: 液面薄膜**正逐渐变连续、光滑**

### 这是第三种对照方案 (与 A1, D7 形成对比矩阵)
- A1: 自反应法 (节点先成膜 12h)
- D7: 标准顺序
- **A8: 单次连续加料** (D9 的 TAPT 版本)

### 预测失败模式
#### 失败 α：**46h 仍粗糙无改善**
- **应对**：
  ```
  - 增加苯胺当量: 8.2 μL → 13.7 μL
  - 延长至 96h 再表征
  - 若仍粗糙, 改为自反应法 (A1 模式)
  ```

#### 成功 β：**72h 连续光滑膜**
- **动作**：72h 取样 XRD/SEM/FTIR
- **下一步**：尝试做 BET 看比表面积是否 >800 m²/g

## D3 (TFPT + TFMB, 0.596, 自反应法)

### 当前已知状态
- 12h 壁上形成**连续、光滑薄膜**（同 A1 模式但用醛节点 TFPT）
- 反复溶解-再形成 (7/6 重启)
- 注：**用了 12.0 M 乙酸（操作错）+ 16.4 μL 苯胺（不匹配 v3.9）**

### 预测失败模式
- 由于 **12M 乙酸操作错**，结果可能不可靠

#### 失败 α：**粉红色油状物大量** (12M 乙酸可能加剧)
- **应对**：
  ```
  - D3 数据仅作参考，不入正式反馈
  - 重新按 v3.9 (6.0M 乙酸, 13.7 μL 苯胺) 重做
  ```

#### 失败 β：**72h 后膜质量可接受**
- **判断**：是 12M 乙酸的错误方法偶然可行？还是自反应法本身有效？
- **应对**：
  ```
  - 必须重做对照实验验证
  - 推荐: 严格按 v3.9 G3-S2 配方 (6.0M 乙酸, 24.6 μL 苯胺)
  ```

## D7 (TFPT + TFMB, 0.596, 标准顺序)

### 当前已知状态
- 72h 后**膜不连续、粗糙的浅黄色固体**
- **粉红色油状物**（与 D3 同类现象）

### 预测失败模式
#### 失败 α：**72h 仍不连续**
- **应对**：
  ```
  - 改加料顺序为 D9 / A8 的"单次连续加"
  - 或改为自反应法 (D3 模式，但修正 6M 乙酸)
  - 苯胺提到 13.7 μL
  ```

#### 失败 β：**出现严重粉红色油状物**
- **应对**：
  ```
  - 乙酸降到 3.0M
  - 加酸时分 2 次 (0.3 mL + 0.3 mL, 间隔 1h)
  ```

## D9 (TFPT + TFMB, 0.596, 新加料顺序)

### 当前已知状态
- 16h 后**白膜连续粗糙**，部分变深红/黑 + 粉红色油状物

### 预测失败模式
#### 失败 α：**膜变黑色**（部分膜已经发生）
- **根因猜测**：氧化或金属离子污染？
- **应对**：
  ```
  - 取变黑部分 FTIR 检查是否亚胺键断裂
  - 若氧化: 重新加氮气保护
  ```

#### 成功 β：**72h 连续光滑白膜**
- **动作**：72h XRD/SEM/FTIR 表征
- **意义**：若成功，确认"单次连续加"优于"标准顺序"和"自反应法"

## 跨实验汇总：3 种加料顺序将构成对照矩阵

| 实验 | 节点 | 加料顺序 | 12h 状态 | 72h 状态 | Class 预期 |
|------|------|----------|----------|----------|------------|
| A1 | TAPT | 自反应法 | 连续光滑 | 连续粗糙 | E (E 倾向) |
| A8 | TAPT | 单次连续加 | n/a | 渐变连续光滑 | F (希望) |
| D7 | TFPT | 标准顺序 | n/a | 不连续粗糙 | E |
| D9 | TFPT | 单次连续加 | n/a | 白膜连续粗糙 | E 倾向 |

**结论期待**（72h 之后做对比）：
- 若 A8 > A1 > D7: TAPT + 单次连续加 是最优
- 若 A1 > A8: 自反应法更优
- 若 D9 > D7: 单次连续加 在 TFPT 也适用
