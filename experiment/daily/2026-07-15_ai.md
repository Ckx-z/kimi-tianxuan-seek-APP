## 关键观察

### "自反应法"通用模式 (A1 / A2 / A5 / D3)
```
节点 (TAPT 或 TFPT) + 溶剂 → 超声 → 120°C × 12h → 壁上形成连续光滑膜
   → 停止反应 → 重启加热 → 膜多次"溶解-再形成"
   → 取管 + 加第二种单体 + 苯胺 + 酸 → 反应 48-72h
```
- 共同问题：**膜不稳定**，会反复溶解-再形成
- 共同观察：12h 形成的初始膜普遍**连续且光滑**（最佳状态）
- 共同观察：加醛后膜质量变粗糙，最终不连续

### "新加料顺序"模式 (A8 / D9)
```
醛 + 苯胺 → 立即加胺 → 最后加酸 (相比标准序的"分两次加" 这是单次连续加)
```
- A8: 46h 后管壁膜**正逐渐变连续、光滑**
- D9: 16h 后**白膜连续粗糙**，部分变深红/黑 + 粉红色油状物

### 标准顺序参照 (D7)
- 72h 后**膜不连续粗糙浅黄色** + 粉红色油状物
- 比"自反应法"早期表现差
